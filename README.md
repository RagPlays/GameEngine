### Input Bindings ### (QUERTY)

#### General Inputs

1. **Reload Scene** (Debug mode only)
    - **Key:** `R`
    - **Action:** Reloads the current scene.

2. **Quit Game**
    - **Key:** `ESC`
    - **Action:** Quits the game.

3. **Next Level**
    - **Key:** `F1`
    - **Action:** Advances to the next level.

4. **Win Level** (Debug mode only)
    - **Key:** `Z`
    - **Action:** Wins the current level.

5. **Kill Player** (Debug mode only)
    - **Key:** `K`
    - **Action:** Kills the player character.

#### Movement Inputs

1. **Move Up**
    - **Key:** `W` or `UP ARROW`
    - **Action:** Moves up.

2. **Move Left**
    - **Key:** `A` or `LEFT ARROW`
    - **Action:** Moves left.

3. **Move Down**
    - **Key:** `S` or `DOWN ARROW`
    - **Action:** Moves down.

4. **Move Right**
    - **Key:** `D` or `RIGHT ARROW`
    - **Action:** Moves right.

#### Action Inputs

1. **Attack**
    - **Key:** `E`
    - **Action:** Perform an attack.

#### Controller Inputs

1. **Move Up**
    - **Button:** `DPAD UP`
    - **Action:** Moves up.

2. **Move Left**
    - **Button:** `DPAD LEFT`
    - **Action:** Moves left.

3. **Move Down**
    - **Button:** `DPAD DOWN`
    - **Action:** Moves down.

4. **Move Right**
    - **Button:** `DPAD RIGHT`
    - **Action:** Moves right.
